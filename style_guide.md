# Style Guide

## Visual Direction
- Light, airy interface with soft shadows and rounded surfaces.
- Calm indigo accents for primary actions and emphasis.
- Neutral slate backgrounds for panels and content blocks.

## Color Palette
- Background: #f7f8fc
- Surface: #ffffff
- Primary: #4f46e5
- Text: #172033
- Muted text: #64748b
- Border: #e5eaf3

## Typography
- Use the existing Geist Sans for headings and body copy.
- Keep labels and metadata concise and readable.
- Prefer medium weight for section titles and semibold for card headings.

## Components
- Buttons: rounded-full, subtle shadow, soft hover transitions.
- Cards: white surface, rounded-2xl corners, gentle elevation.
- Inputs: light slate fill, rounded-xl corners, simple borders.
- Dialogs: airy spacing, light shadow, calm borders.

## Motion
- Use short, smooth transitions for hover and focus states.
- Keep interactions subtle rather than flashy.

## Layout Principles
- Keep spacing generous and consistent.
- Use the board as the main focus with clear separation between columns.
- Prioritize clarity and readability over dense UI.
